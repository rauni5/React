# React Chat Application

## Overview

This project was developed to learn React fundamentals, component-based architecture, state management, and modern frontend development practices.
The application simulates a simple chat interface where users can send and display messages dynamically.

## Features

* Component-based design
* Interactive chat interface
* Dynamic message rendering
* State management using React
* Responsive layout
* Modern React development workflow

## Technologies Used

* React
* JavaScript (ES6+)
* Vite
* CSS

## Project Structure

src/
├── components/
├── assets/
├── App.jsx
└── main.jsx

## Concepts Practiced

* Functional Components
* Props
* State Management
* Event Handling
* Component Reusability
* Project Structure Organization

## Installation

```bash
git clone <repository-url>
cd react-project
npm install
npm run dev
```
